## paper notes

Add paper notes as markdown files in `src/tldr`. The route is `/tldr`, with
individual notes at `/tldr/<filename>`.

Use this frontmatter shape:

```yaml
---
title: "My notes on Paper Title"
paperTitle: "Paper Title"
paperUrl: "https://arxiv.org/abs/0000.00000"
date: 2026-06-12
topics: [topic-one, topic-two]
area: "research area"
description: "Short summary of the note."
---
```
